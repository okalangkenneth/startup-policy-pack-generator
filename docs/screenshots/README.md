# Screenshots

## How to Add Screenshots

1. **Open the application**: Navigate to http://localhost:5175 or open the built `dist/index.html`

2. **Take Main Form Screenshot**:
   - Fill in some sample data in the form
   - Capture the full page showing the header and form sections
   - Save as `main-form.png` in this directory
   - Recommended size: 1200x800 or similar

3. **Take ZIP Output Screenshot**:
   - After generating a ZIP, show the downloaded file or success state
   - You can also show the extracted ZIP contents (the 7 .md files)
   - Save as `zip-output.png` in this directory
